  1. DON’T use section to divide content from the header and footer; use div instead (see the doctor)
  2. DON’T use section to wrap a tab switcher for DOM manipulation or styling
  3. DON’T use section for sidebar or other tangentially-related content boxes; use aside instead
  4. DON’T use section just to add a border or drop shadow around something; use div instead
  5. DON’T use section for the wrapper when implementing faux columns; again, use div instead
  6. DON’T use section to nest elements when trying to avoid IE6’s float double-margin bug (or a similar layout-related issue); again, use div
  7. DON’T use section to hold an individual author bio on a blog post or news article; use aside instead
